## Run locally

### Prerequisites

- php,composer

- docker or docker desktop

- aws cli

Launch app 

```bash
php artisan serve
```

launch the bundler(vite)

```bash
pnpm dev
# npm run dev
# yarn dev
```

and launch localstack for using aws service  locally

start container

```bash
docker compose up
```

stop container

```bash
docker compose stop
```

delete container

```bash
docker compsoe down
```
